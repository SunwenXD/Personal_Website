## 專案概述

這是一個整合了物聯網 (IoT) 與物聯網 (IoT) 概念的智慧執法輔助系統。本專案前端採用 **Flutter** 實現跨平台行動應用程式，內嵌 **YOLO** 模型於邊緣設備（手機），讓使用者能夠執行在地化物件偵測，即時捕捉違規停車畫面並自動擷取關鍵證據。

後端利用 **FastAPI** 處理高併發影像傳輸，並透過 **OCR** 技術識別車牌號碼與地理位置資訊，將結構化數據儲存於 **PostgreSQL**。管理者或執法人員可透過基於 **Vue.js** 建構的後台儀表板（Dashboard），結合 **Leaflet** 地圖元件，直觀地查看違規熱點、歷史紀錄與空間分佈，從而提升管理效率。

## 使用技術

- **Frontend (Mobile)**: **Flutter** — 建構跨平台行動介面
- **Frontend (Web)**: **Vue.js** + **Tailwind CSS** — 建構響應式管理後台
- **AI / CV**: **YOLO** (物件檢測) + **OCR** (車牌辨識) — 實現 Edge AI 與影像處理
- **Backend**: **FastAPI** — 高效非同步 API 傳輸
- **Data & Ops**: **PostgreSQL** (空間資料庫), **Docker** (容器化部署)
- **Mapping**: **Leaflet** — 即時地圖視覺化與空間分析

## 核心功能

1. **Edge AI 即時感知**：在手機端直接執行 YOLO 推論，無需依賴遠端伺服器進行影像判讀，實現低延遲的違規偵測。
2. **智能證據擷取**：自動裁切違規車輛影像，並串接 OCR 引擎強化模糊影像與車牌辨識，過濾無關雜訊。
3. **自動化工單流程**：從拍攝、識別到上傳資料庫（儲存時間、地點、照片）皆為自動化流程，減少人工輸入錯誤。
4. **即時空間視覺化**：整合 Leaflet 地圖，將違規事件以地理資訊形式呈現，支援區域篩選與熱點統計。
5. **自動化通知機制**：資料入庫後觸發通知系統，即時將違規資訊傳遞至相關人員（檢舉人或被檢舉車主），形成完整閉環。

## 成果

- **實現端邊雲完整架構**：成功打造從「邊緣端 AI 辨識 (Edge)」、「高速通訊傳輸 (Network)」到「雲端資料視覺化 (Cloud)」的完整全端解決方案，展現系統整合能力。
- **高效能的邊緣運算應用**：透過將 YOLO 模型部署於手機端，有效解決了傳統架構需傳輸大量影像至伺服器造成的頻寬浪費與延遲問題，保護了使用者隱私。
- **資料驅動的空間分析能力**：結合 PostgreSQL 空間處理與 Leaflet 地圖視覺化，成功將非結構化的違規影像轉化為結構化的地理資訊數據，可輔助執法單位進行統計與防範分析。
- **全流程自動化**：極簡化了使用者操作步驟（拍攝即完成所有資料整理與上傳），提升了違規檢舉的效率與準確度。
