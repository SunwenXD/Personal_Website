## 專案概述

 幫助使用者下載yt廣告，並且無須下載flutter就能使用，不會汙染環境

## 使用技術

- **yt-dlp** — 下載影片
- **flutter** — gui介面
- **fastapi** - 跟後端交互

## 核心功能

1. 輸入影片網址/影片清單，以列表呈現給用戶
2. 下載有特殊樣式
3. 切換下載風格
4. 記住使用者偏好

## 成果

- **零環境依賴與安全高效**：整合 FastAPI 與 Flutter 技術，提供「開箱即用」的下載體驗。使用者無需安裝 Python 或建構開發環境，完全隔離系統汙染，同時避免第三方風險網站的廣告阻擋與限速問題。
- **優雅的圖形化介面 (GUI)**：將傳統 Command Line 的複雜參數轉換為現代化的 Flutter 響應式介面，支援直觀的操作流程與即時進度回報，讓影音管理比單純依靠指令更輕鬆直觀。
- **穩健的核心解析能力**：依托 **yt-dlp** 強大的社群維護版核心，有效克服網頁端下載插件易失效、卡頓或解析度受限的問題，順利支援播放清單批次處理。
- **個人化與隱私保護**：內建使用者偏好記憶功能，自動保留選定的解析度與儲存路徑。所有下載動作均於本地端 (Localhost) 完成，確保帳號 Cookie 與個資絕對安全，不被第三方擷取。
